# 退職シミュレーターの画面を表示する手順

## ステップ1: ターミナルを開く

1. Cursor の画面上部メニューで **「ターミナル」** をクリック
2. **「新しいターミナル」** をクリック  
   （またはキーボードで **Ctrl + `** ［Macなら **Command + `**］を押す）
3. 画面下（または横）に黒い枠（ターミナル）が開く

---

## ステップ2: npm の権限エラーを直す（一度だけ）

ターミナルに **そのまま** 次のコマンドをコピーして貼り付け、**Enter** を押す。

```bash
sudo chown -R $(whoami) ~/.npm
```

- パスワードを聞かれたら、Macのログインパスワードを入力（画面には何も表示されません）
- 入力し終えたら **Enter** を押す

---

## ステップ3: プロジェクトフォルダに移動

ターミナルに次の **1行** をコピーして貼り付け、**Enter** を押す。

```bash
cd /Users/sasadakota/taishoku-simulator
```

---

## ステップ4: パッケージをインストール

次の **1行** をコピーして貼り付け、**Enter** を押す。

```bash
npm install
```

- 数十秒〜1分ほどかかることがある
- 最後にエラーが出ず、またプロンプト（`$` やパス）が戻ればOK

---

## ステップ5: 開発サーバーを起動

次の **1行** をコピーして貼り付け、**Enter** を押す。

```bash
npm run dev
```

- **「Ready in ...」** や **「Local: http://localhost:3000」** と表示されれば起動できている

---

## ステップ6: ブラウザで画面を開く

1. **Chrome / Safari / Edge** など、普段使っているブラウザを開く
2. アドレスバー（URLを入力する欄）をクリック
3. 次のアドレスを **そのまま** 入力する  
   **http://localhost:3000**
4. **Enter** を押す

→ 退職シミュレーターの画面が表示されます。

---

## 止めるとき

- ターミナルで **Ctrl + C**（Macなら **Control + C**）を押すと、開発サーバーが止まります。
- もう一度見たいときは、もう一度 **ステップ5** の `npm run dev` から実行すればOKです（ステップ2〜4は不要）。

---

## うまくいかないとき

- **「command not found: npm」** と出る  
  → [Node.js](https://nodejs.org/) をインストールしてから、ステップ3以降をやり直す。

- **「EADDRINUSE」** や **「port 3000 is already in use」** と出る  
  → すでに別のアプリが 3000 番を使っています。そのアプリを終了するか、ターミナルで `npm run dev` を止めてから再度実行する。

- **ブラウザで「接続できません」**  
  → ターミナルで `npm run dev` が動いたままか確認する。止まっていたらもう一度 `npm run dev` を実行する。
