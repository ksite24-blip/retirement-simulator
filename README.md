# 退職シミュレーター

有給残日数・給与・ボーナス月を入力し、最短退職日と損得をシミュレートするWebアプリです。

## 技術スタック

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript

## 使い方

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 入力項目

- **有給残日数** … 残っている有給休暇の日数
- **給与（月収）** … 月給（円）
- **ボーナス月** … ボーナスが支給される月をカンマ区切り（例: 6, 12）

## 表示内容

- **最短退職日** … 退職予告30日後を想定した日付
- **想定収入** … 最短で退職した場合の勤務分＋有給買取の目安
- **損得** … ボーナス月まで在籍した場合との差（損の場合は△表示）

※試算は目安です。実際の金額・退職条件は就業規則に従ってください。
